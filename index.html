We are VScanner.
Vulnerability scanner device.

A small device with a big ability for security testing. It's currently geared towards web security. It runs security tests and manages a lot of well-known security tools such as (OpenVas, Wfuzz, SQLMap, DNS recon, robot analyzer...) take their results, feedback to the rest of tools and merge the results. All in simple one click.

Target audience is small and medium sized companies, who are running their business with online web application and using the internet for their work.
