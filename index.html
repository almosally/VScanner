Main tools run by the device

Nikto

Web server scanner which performs comprehensive tests against web servers for multiple items, including over 6700 potentially dangerous files/programs.

Nmap

Utility for network discovery and security auditing.

OpenVAS

Full-featured vulnerability scanner. Its capabilities include unauthenticated testing, authenticated testing, various high level and low level Internet and industrial protocols.

The Harvester

The objective of this program is to gather emails, subdomains, hosts, employee names, open ports and banners from different public sources like search engines.

Sqlmap

Testing tool that automates the process of detecting and exploiting SQL injection flaws and taking over of database servers. It comes with a powerful detection engine.

Wfuzz

It has been created to facilitate the task in web applications assessments and it is based on a simple concept: it replaces any reference to the FUZZ keyword by the value of a given payload.
